# Multilingual Sentiment Analysis (Indian Languages)

## Description
This project implements a multilingual rule-based sentiment analysis system for Indian languages. The system automatically detects the input language and classifies sentiment as Positive, Negative, or Neutral using language-specific sentiment dictionaries.

It supports multiple regional Indian languages, making the solution relevant for multilingual real-world NLP applications in India.

---

## Problem Statement
Most sentiment analysis systems are designed primarily for English text and do not support regional Indian languages effectively. Given the large amount of multilingual content generated in India, there is a need for sentiment analysis systems that can process and classify sentiment in Indian languages.

---

## Objective
- Automatically detect the language of input text  
- Support sentiment analysis for multiple Indian languages  
- Classify text as Positive, Negative, or Neutral  
- Demonstrate multilingual NLP relevance for Indian use cases  

---

## Supported Languages
- Hindi  
- Telugu  
- Tamil  
- Kannada  
- Malayalam  

---

## Methodology
1. User enters text in any supported Indian language  
2. System detects language using language-specific keywords  
3. Input text is preprocessed by lowercasing and punctuation removal  
4. Phrase-based sentiment matching is performed using language dictionaries  
5. Positive and negative match counts are compared  
6. Final sentiment label is returned  

---

## NLP Techniques Used

- Text Preprocessing  
- Phrase Matching  
- Rule-Based Sentiment Classification  
- Multilingual Lexicon Analysis
